# 📊 Customer Churn Analysis & Prediction
A data analytics project that analyzes telecom customer churn and predicts customers who are likely to leave using **Python, SQL, and Machine Learning**.

The project demonstrates a **complete analytics workflow** from data cleaning to business insights and dashboard visualization.

---

## 🧠 Problem

Customer churn reduces recurring revenue and increases acquisition costs.

This project answers:

- Which customers are likely to churn?
- What factors influence churn?
- How can businesses detect churn early?

---

## 📁 Dataset

**IBM Telco Customer Churn Dataset**

- 7,043 customers  
- 21 features including customer behavior, services, billing, and churn status.

Key fields:

| Feature | Description |
|------|------|
Tenure | Months with the company |
Contract | Contract type |
MonthlyCharges | Monthly bill |
PaymentMethod | Payment type |
Churn | Customer left or not |

---

## 🔑 Key Insights

- **Month-to-month contracts** show highest churn
- Customers within **first year** are more likely to leave
- **Electronic check payments** correlate with higher churn

---

## 💡 Business Recommendations

- Offer retention incentives during early customer lifecycle
- Encourage **auto-payment methods**
- Monitor **high-risk customers using churn prediction**
